# Python Budget Analyzer

A Python tool that analyzes personal transaction data and visualizes spending across 
needs, wants, and savings using the 50/30/20 budgeting framework. Outputs a printed 
summary of percentage allocations and generates a pie chart and bar chart of spending patterns.

## Requirements
pip install pandas matplotlib

## How to run
1. Clone the repo
2. Run budget.py
3. Sample transaction data is included (sample_transactions.csv)

## Built with
- Python
- Pandas
- Matplotlib
